# HoverLinkSource

## Properties

### display

```ts
display: string
```

显示在“页面预览”插件设置中的字段。该字段应与您插件的名称一致。

### defaultMod

```ts
defaultMod: boolean
```

`hover-link` 事件是否需要按下 'Mod' 键才可以触发。
