# HoverLinkSource

## Properties

### display

```ts
display: string
```

The string that will be displayed in the 'Page preview' plugin settings. It should match your plugin's display name.

### defaultMod

```ts
defaultMod: boolean
```

Whether or not the `hover-link` event requires the 'Mod' key to be pressed to trigger.
