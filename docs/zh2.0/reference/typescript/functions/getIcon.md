# getIcon

```ts
export function getIcon(iconId: string): SVGSVGElement | null;
```

Create an SVG from an iconId. Returns null if no icon associated with the iconId.

## Parameters

| Parameter | Description |
|-----------|-------------|
| `iconId` | the icon ID |
