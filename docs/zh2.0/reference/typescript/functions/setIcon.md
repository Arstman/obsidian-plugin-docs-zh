# setIcon

```ts
export function setIcon(parent: HTMLElement, iconId: string, size?: number): void;
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `parent` | 插入 icon 的 HTML 元素 |
| `iconId` | icon 的 ID |
| `size` | 设置宽高的像素值，默认为 16 |
