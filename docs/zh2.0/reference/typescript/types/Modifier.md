# Modifier

```ts
export type Modifier = 'Mod' | 'Ctrl' | 'Meta' | 'Shift' | 'Alt';
```

Mod = MacOS 上的 Cmd 键 / 其他系统上的 Ctrl 键
Ctrl = 任意系统的 Ctrl 键
Meta = MacOS 上的 Cmd 键 / 其他系统上的 Win 键
